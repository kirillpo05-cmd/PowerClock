Формат: имя поля, тип, ограничения, пример значения. «Без изменений» — тоже ответ.

## API

формат ответа, поведение при ошибке. «Нет» — тоже ответ (фронт не ходит в сеть).

## Экраны и компоненты

Где в layout, что отображает, действия пользователя, все 4 состояния
(loading / error / empty / success). Для графиков: правила кодирования
(длина/цвет/угол), легенда, tooltip-формат.

## Бизнес-логика

Формулы, валидации, пороги, переходы состояний. Числами: «скрыть карточку, если
заполнено < 50% ячеек», а не «если данных мало».
